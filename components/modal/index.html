Modal

Modals add dialog and content via a secondary window but allow the user to remain connected to the context of their task.

Note on usage

Modals may interupt the flow of a task make sure the choice to use a modal fits in as a secondary task or a deeper dive into the information displayed

Before getting started with Bootstrap's modal component, see what has changed.

getbootstrap.com/docs/4.0/components/modal/

Modal drawers #

We have extended Bootstraps core modal to slide in as a drawer. Consider using this for some supplementary content. Add the following modifier classes to the same div as the .modal class, to slide in from the left .modal-slide-left or to slide in from the right .modal-slide-right.

Optional sizes #

Modals have three optional sizes, available via modifier classes to be placed on a .modal-dialog. These sizes kick in at certain breakpoints to avoid horizontal scrollbars on narrower viewports.

Size Class Modal max-width
Small .modal-sm 375px
Default None 640px
Large .modal-lg 900px
Extra large .modal-xl 1140px

Our default modal without modifier class constitutes the "medium" size modal.
