Datepicker

Datepickers help with date selection.

We mostly use an input group with a calendar icon button that launches the datepicker in a popup, but the datepicker can also be used inline.

Input Group

Only use placeholders when it is clear the placeholder will not be confused with the expected input. Placeholders may cause confusion as the input field may be viewed as already filled.


<div class="input-group mb-3">
  <input type="text" class="form-control" placeholder="mm/dd/yyyy" aria-label="Datepicker" aria-describedby="button-datepicker">
  <div class="input-group-append">
    <button class="btn btn-outline-secondary" type="button" id="button-datepicker"><span class="t-calendar"</span></button>
  </div>
</div>
