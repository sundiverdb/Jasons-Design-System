Components

Radio Button

Use radio buttons to display a list of options where the user may only select a single option.

Example

Default radios are improved upon with the help of .form-radio.

Disabled radios are supported, make sure to add the .disabled class to the <label> this will apply a lighter color to help indicate the input's state.

Default (stacked)

By default, any number of radios that are immediate sibling will be vertically stacked and appropriately spaced with .form-radio.

Try and design without using disabled states. Disabled elements may cause confusion when the reason for the element being disabled isn't clear. Provide a clear validation reason when disabling an element.

<div class="form-radio">
  <input class="form-radio-input" type="radio" name="exampleRadios" id="exampleRadios1" value="option1" checked>
  <label class="form-radio-label" for="exampleRadios1">
    Default radio
  </label>
</div>
<div class="form-radio">
  <input class="form-radio-input" type="radio" name="exampleRadios" id="exampleRadios2" value="option2">
  <label class="form-radio-label" for="exampleRadios2">
    Second default radio
  </label>
</div>
<div class="form-radio disabled">
  <input class="form-radio-input" type="radio" name="exampleRadios" id="exampleRadios3" value="option3" disabled>
  <label class="form-radio-label disabled" for="exampleRadios3">
    Disabled radio
  </label>
</div>

Inline #

Group radios on the same horizontal row by adding .form-radio-inline to any .form-radio.

<div class="form-radio form-radio-inline">
  <input class="form-radio-input" type="radio" name="inlineRadioOptions" id="inlineRadio1" value="option1">
  <label class="form-radio-label" for="inlineRadio1">1</label>
</div>
<div class="form-radio form-radio-inline">
  <input class="form-radio-input" type="radio" name="inlineRadioOptions" id="inlineRadio2" value="option2">
  <label class="form-radio-label" for="inlineRadio2">2</label>
</div>
<div class="form-radio form-radio-inline">
  <input class="form-radio-input" type="radio" name="inlineRadioOptions" id="inlineRadio3" value="option3" disabled>
  <label class="form-radio-label" for="inlineRadio3">3 (disabled)</label>
</div>

Radio Switches #

These differ from the Switches and Checkbox components in that they may have more than two options but only a single option can be selected, and the options are not simply toggled on or off.

This format should be used when you need radio buttons distinguished from standard or inline radio buttons.

<div class="form-radio-switch">
  <input type="radio" name="radio-one" value="yes" checked>
  <label for="radio-one">Yes</label>
  <input type="radio" name="radio-two" value="no" >
  <label for="radio-two">No</label>
</div>

<label class="d-block" >Three fields or more.</label>
<div class="form-radio-switch">
  <input type="radio" name="radio-three" value="one" checked>
  <label for="radio-three">One</label>
  <input type="radio" name="radio-four" value="two" >
  <label for="radio-four">Two</label>
  <input type="radio" name="radio-five" value="three" >
  <label for="radio-five">Three</label>
</div>
