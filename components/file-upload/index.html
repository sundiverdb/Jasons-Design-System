File Upload

Allow the user to upload a file or submit content of their own with a file browser or custom drag and drop field.

File browser

Input groups include support for custom file inputs. Browser default versions of these are not supported.

<div class="mb-3">
  <label for="formFile" class="form-label">Default file input example</label>
  <input class="form-control" type="file" id="formFile">
</div>

Drag and Drop

This is a how our drag and drop element should appear. On the left is the static state. On the right is the active state. The active state appears when something is being dragged into/over the area.

Files should be automatically uploaded and processed after being selected or dropped into the target area.

Normal display
Files must be in PDF, PNG, JPG, or GIF format and cannot exceed 15mb in size and can't be a secure PDF.
Active display
Files must be in PDF, PNG, JPG, or GIF format and cannot exceed 15mb in size and can't be a secure PDF.
