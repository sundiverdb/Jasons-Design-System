Object Styles

Specific shapes or effects can help establish hierarchy on the page and give cues about interaction.

Borders

Border-Radius

Add classes to an element to easily round its corners.

For no border-radius use the .rounded-0 class

For a border-radius of .25rem use the .rounded class

For a border-radius of .5rem use the .rounded-lg class

For a border-radius of 1rem use the .rounded-xl class

Border-Colors

The classes are named using the format border-{name}.

Shadows #

By default most components do not have shadows. But you can quickly add or remove shadows with the following utility classes.

No shadow - .shadow-none
Small shadow - .shadow-sm h-offset=0 y-offset=2px blur=4px color=rgba(0, 0, 0, .06)
Regular shadow - .shadow h-offset=0 y-offset=4px blur=8px color=rgba(0, 0, 0, .08)
Large shadow - .shadow-lg h-offset=0 y-offset=16px blur=32px color=rgba(0, 0, 0, .10)
