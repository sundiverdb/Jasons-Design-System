Iconography

We use icons to visually communicate information in a distinct and precise manner.

Tyrian Icons

Our primary icons come from our custom Tyrian icon font. You will need to make sure they are compiled in your css file and the requisite font files are included. The classes use the pre-fix t-{icon name}.

To use the Tyrian icon font in your design application download and install the font on your machine.
Copy and paste the glyph() just to the right of the icon name, to use it in your applications.

Our fallback icons come from FontAwesome. You can place Font Awesome icons just about anywhere using the CSS Prefix fa and the icon's name. Font Awesome is designed to be used with inline elements (Use the <span> tag to be semantically correct).
