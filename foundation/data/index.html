Foundation

Data Visualization

The graphical representation of data to simplify complex concepts and enable understanding.

Charts

Charts are not just decorative, they should enable exploration, montitoring, and provide explanation. Charts should be simple and represent data in a way that improves its consumption.

Types of charts

Bar

Each bar is mapped to a value with a length determined by the metric. Bar charts may be displayed vertically or horizontally.

Stacked Bar

Stacked bar charts use color to show additional dimensions as parts of a whole. No more than six categorical colors should be used.

Line

Use to compare many singular or similar data points plotted on a line that change over time. Helpful in displaying data chronologically.

Area

Use to compare many similar data points within a time frame where seeing the overlap of the data is important.

Pie

Divided segments of an arc to show the proportional value of each piece of data.Use to show a few large segments of a total set.

Donut

Show smaller segments in a total set or to show progress in smaller areas, also an additional data point or icon may be presented in the center.

Usage #

Categorical colors

Categorical colors are ordered but may be used out of order if necessary, use them for categorical scales. Do not use them for ordinal, interval, or ratio scales.

Example: locations (e.g., cities, states) or names (e.g., employee, company)
Sequential colors

Sequential colors are ordered. Use these for ordinal and interval scales. It's also acceptable to use these for ratio scales. Do not use these for categorical scales.

Example: ranked list (e.g., 1st, 2nd, 3rd), or (e.g., 2005, 2010, 2015, 2020)
Diverging colors

Diverging colors are ordered. Use these for ordinal and ratio scales, especially when there is a meaningful middle value. These may also be used for interval scales. Do not use these for categorical scales.

Example: ratios (e.g., low, neutral, high), (e.g., -20, -10, 0, 10, 20)

Be consistent

With color when there are multiple charts in the same view or presenting similar context. If a color is used to represent something in one chart, all other charts should have that same color relationship.
